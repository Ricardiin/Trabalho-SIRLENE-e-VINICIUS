# 🛍️ Sistema de Gerenciamento de Produtos

Este projeto é um sistema web desenvolvido para gerenciamento de produtos, permitindo realizar operações completas de cadastro, edição, exclusão e visualização de itens em estoque.

O sistema foi desenvolvido como parte de uma atividade, com foco em integração entre frontend e backend, utilizando tecnologias Java Web.

---

## 🚀 Funcionalidades

- 📦 Cadastro de produtos  
- ✏️ Edição de produtos  
- ❌ Exclusão de produtos  
- 📋 Listagem de produtos  
- 👁️ Preview em tempo real no cadastro/edição  
- 📊 Geração de relatório em PDF  
- 🔐 Controle de acesso (modo gerente e usuário)  

---

## 🛠️ Tecnologias Utilizadas

### 🔙 Backend
- Java  
- Servlet (Jakarta)  
- JDBC  
- MySQL  

### 🎨 Frontend
- JSP  
- HTML5  
- CSS3  
- Bootstrap 5  
- JavaScript  

### 📄 Outros
- iText PDF (geração de relatórios)  

---

## 🧠 Conceitos Aplicados

- Arquitetura MVC (Model-View-Controller)  
- CRUD completo (Create, Read, Update, Delete)  
- Manipulação de sessões (login e permissões)  
- Integração entre frontend e backend  
- Geração de documentos PDF dinâmicos  

---

## 📈 Objetivo do Projeto

Este projeto tem como objetivo praticar e demonstrar conhecimentos em desenvolvimento web com Java, incluindo integração completa entre frontend e backend, manipulação de dados e geração de relatórios.
